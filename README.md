A  Social Platform (Work in Progress)

This project is a full-stack web application designed as a distraction-free, text-first social platform. It focuses on meaningful interaction, idea sharing, and personal growth rather than short-form, addictive content.

---

## Core Features

### Content Creation

Users can create three types of content:

* Blogs – Long-form, thoughtful posts
* Ideas – Collaborative concepts that others can contribute to
* Diary Entries – Personal reflections (private by default, optional public sharing)

---

### Idea Collaboration

* Users can post ideas they want to build
* Other users can connect and express interest in collaborating
* Designed to help people find teammates and turn ideas into real projects

---

### Social Interactions

* Like posts
* Comment on posts
* Follow other users
* View user profiles and their content

---

### Feed System

* Public feed showing latest content
* Filter by:

  * Blogs
  * Ideas
  * Public diaries
* Chronological feed without algorithmic manipulation

---

### Authentication

* Implemented using Clerk
* Secure user authentication and session management
* Webhook-based synchronization with MongoDB for storing user data

---

### Personal Growth Focus

* Private diary system encourages reflection and self-improvement
* Optional sharing allows users to engage with the community

---

## Tech Stack

* Frontend: Next.js, React, Tailwind CSS
* Backend: Next.js API Routes
* Database: MongoDB with Mongoose
* Authentication: Clerk

---

## Status

This project is currently a work in progress. Core features such as authentication, content creation, and feed are implemented, with collaboration and interaction systems under development.

---

## Vision

To create a platform where users can share ideas and thoughts without distraction, collaborate on meaningful projects, and grow through consistent writing and reflection.
